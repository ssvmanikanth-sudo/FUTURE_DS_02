# FUTURE_DS_02
📊 Customer Churn & Retention Analysis (Power BI)
📌 Project Overview
This project focuses on analyzing customer data from a subscription-based business to identify churn patterns, retention trends, and customer lifetime behavior.

The goal is to derive actionable insights that help reduce customer churn and improve retention strategies using Power BI.

🎯 Objectives
Identify customers who are likely to churn
Analyze retention trends over time
Perform cohort analysis to track customer behavior
Calculate customer lifetime metrics (CLV, tenure)
Provide data-driven recommendations to improve retention
🛠️ Tools & Technologies
Power BI – Data visualization & dashboard creation
DAX (Data Analysis Expressions) – Calculated colum
